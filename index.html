HTML

HTML is a language of tags that use the Internet to host web pages. Browsers support HTML documents from a web server or local memory and transfer documents to multimedia web pages. HTML describes the structure of a web page semantically and initially with hints for displaying the document.

CSS

CSS is a special page style language used to describe their appearance. The pages themselves are written in markup languages. CSS is the core technology of the World Wide Web, along with HTML and JavaScript.

JavaScript

JavaScript is a dynamic, object-oriented prototype programming language. Implementation of the ECMAScript standard. Use is used to create web page scripts that allow the client (end-user device) to interact with the user, control the browser, asynchronously communicate with the server, change the structure and appearance of the web page.
